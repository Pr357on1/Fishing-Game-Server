# Island Fishing Game

A beautiful, minimalistic fishing game set on a tropical island with smooth animations and an intuitive inventory system.

## Features

- **Fishing Minigame**: Catch fish using a 1D timing-based minigame
- **Inventory System**: Collect and manage your fish with a hotbar for quick access
- **Shop System**: Buy upgrades and sell your catch to the shop keeper
- **Island World**: Explore a tropical island with animated palm trees
- **Camera System**: Smooth camera that follows your character
- **Fish Rarities**: Common, Uncommon, Rare, Epic, and Legendary fish

## How to Play

### Controls
- **A/D** or **Left/Right Arrow Keys**: Move left/right (1D movement)
- **W** or **Space**: Jump
- **F**: Start fishing
- **Space**: Catch fish (during fishing minigame) or Jump (when not fishing)
- **E**: Interact with shop keeper (when nearby)
- **I**: Open inventory
- **1-5**: Select hotbar slots

### Fishing
1. Press **F** to start fishing
2. Watch the moving indicator on the fishing bar
3. Press **Space** when the indicator is in the green target zone
4. Successfully catching fish rewards you with a random fish based on rarity

### Shop
1. Walk up to the blue shop keeper (marked with "Shop")
2. Press **E** to open the shop
3. Buy fishing rods and bait, or sell your fish for money

### Inventory
1. Press **I** to open your inventory
2. Click on items to add them to your hotbar
3. Use number keys 1-5 to quickly select hotbar items

## Fish Rarities

- **Common** (50% chance): Tuna, Salmon, Bass - $10-15
- **Uncommon** (25% chance): Pufferfish, Swordfish - $30-40
- **Rare** (15% chance): Shark, Stingray - $80-100
- **Epic** (8% chance): Golden Fish, Rainbow Trout - $200-250
- **Legendary** (2% chance): Legendary Leviathan - $1000

## Running the Game

Simply open `index.html` in a modern web browser. No build process or server required!
